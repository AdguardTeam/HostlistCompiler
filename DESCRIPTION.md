Adblock/AdGuard Hostlist & Rules Compiler — run locally, via CLI, or as a Compiler-as-a-Service deployed on Cloudflare Workers, Deno Deploy, Vercel Edge Functions, or AWS Lambda@Edge. Aggregates and compiles host blocklists from disparate sources, with AGTree-powered AST parsing, multi-syntax detection and translation across AdGuard, uBlock Origin, and Adblock Plus formats, and 11 built-in transformations. Compatible with Pi-hole, AdGuard, AdGuard Home, eBlocker, uBlock Origin, Brave (aggressive mode), and AdNauseam.
