Frontend Framework Proofs of Concept

Explore how the Adblock Compiler frontend can be implemented in two popular JavaScript frameworks. Each PoC demonstrates routing, state management, theme toggling, and API integration. These implementations are incubator-stage experiments — not yet part of the production UI.

💚 Vue 3 CDN

Vue 3 with Vue Router 4 and the Composition API. Demonstrates named routes, route parameters, navigation guards, and two-way data binding.

  • Vue 3
  • Vue Router 4
  • Composition API
  • v-model

How to run

cd poc/vue python3 -m http.server 8001
Open Vue PoC →
🅰️ Angular Build Required

  • TypeScript
